
# HTML Tag extractor
```
TO extract html tags from a giving URL or HTML page files.
```

## Usage:

```shell
  extractor <URL>
  extractor -f <FileName>
  extractor -d <DirectoryName>
```

## Ex:

```shell
  extractor -d ./all_html_files
  
  # Same as:
  
  cd ./all_html_files
  extractor -d .
```
